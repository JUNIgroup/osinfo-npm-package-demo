
## Usage

Get system info object:
```sh
    osinfo.info
```

Log basic system info to console:
```sh
    osinfo.log()
```

### Example:
```sh
    // Require
    const osinfo = require('osinfo-j')

    // Get system info object
    console.log( osinfo.info )

    // Log basic system info
    osinfo.log()
```
